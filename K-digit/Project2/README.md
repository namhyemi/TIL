# 데이터 시각화 프로젝트 기획안

기획안 작성일자 : 2020/08/08



### 팀명

맛있는거 4먹조



### 프로젝트 주제 및 개요 

카카오 맵 리뷰 데이터를 통한 맛집 추천



### 프로젝트 수행 방향 

- 프로젝트 진행 방향
  - 가게 정보 및 리뷰 데이터 수집
  - 데이터 베이스 구축
  - 데이터 분석
    - 데이터 시각화
    - 텍스트 감성 분석
  - 대시보드 작성
  
  
- 수행도구, 데이터 소개
  - 데이터 수집 및 분석 : Python
  - 데이터 저장 : MySQL
  - 데이터 처리 : Spark -> Python (시간 관계상 변경)



### 프로젝트 조직 (구성원 및 역할)
