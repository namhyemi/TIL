## SQL

DDL : Data Definiton Language - 데이터 정의 언어

- CREATE
- DROP
- ALTER

DML : Data Manipulation Language - 데이터 조작 언어

- INSERT
- UPDATE
- DELETE
- SELECT

DCL : Data Control Language -  데이터 제어 언어

- GRANT
- REVOKE
