# typora 기본 사용법

**typora는 개발자들의 통일된 문서 양식 이다. (확장자 : md)**



1. 제목 : `#` 의 개수로 설정(Ctrl + 숫자)
2. 내용  
3. 코드 
4. 인라인  
5. 코드블럭 : `'''`  사용. 코드 블럭 생성, 언어 선택으로 가시성 높임
6. 목록 :  `-`  사용 (`숫자.` 는 ordered list)
   - `tab 키` 사용시 추가 들여쓰기 가능
7. 표 : `|` 사용 (Ctrl+enter 행 추가)
8. 수식블럭 : `$$`, Latex사용 
   1.  이모티콘 :  `: '감정'`  ex) :smile:
9. `ctrl+B` : 굵은 글씨 생성
